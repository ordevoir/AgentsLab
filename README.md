# agentslab

Лаборатория для экспериментов с RL и MARL на базе TorchRL.
- Поддержаны Gymnasium, PettingZoo/MPE2 и VMAS.
- Примеры: DQN (одноагентный), PPO (мультиагентный, VMAS).

## Установка
```bash
pip install -e . -r requirements.txt
```

## Быстрый старт (из блокнота)
См. `notebooks/DQN_CartPole.ipynb` и `notebooks/MARL_VMAS_PPO.ipynb`.
